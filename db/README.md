# Database

- `migrations/001_init.sql` — full molecule graph + org hierarchy + RAG chunks (pgvector).
- `seed/antibiotics.json` — generated by `content/pipeline/generate-seed.mjs`.

Apply migrations against Supabase/Postgres when `DATABASE_URL` is available:

```bash
psql "$DATABASE_URL" -f db/migrations/001_init.sql
```

Local development uses the JSON seed via the API in-memory store (no DB required).
