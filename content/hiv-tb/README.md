# HIV / TB authoring set

SA-critical chronic area (Doc 29). ART and TB regimens are clinician-directed — Materia teaches the why, never invents a prescription.

## Rules

1. Clinical dosing / regimen tables stay **draft** until founder pharmacist review against current SA ART & TB guidelines.
2. Attach a real `Source` per clinical field before publish.
3. Never copy SAMF / MIMS / Lexicomp.
4. Interaction flags (e.g. rifampicin ↔ dolutegravir) are educational — action text must refuse invented dose changes.
